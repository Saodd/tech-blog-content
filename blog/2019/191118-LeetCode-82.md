## 读题

```text
给定一个排序链表，删除所有含有重复数字的节点，只保留原始链表中 没有重复出现 的数字。

示例 1:

输入: 1->2->3->3->4->4->5
输出: 1->2->5

示例 2:
```

## 分析

思路就是，先遍历一遍，找出所有重复的数值；然后再遍历一遍，把重复的删除掉。

至于用什么数据结构来保存重复值，首先能想到`map`；但是感觉map好像又比较臃肿，其实自己实现一个红黑树会比较好吧。

## 实现

```go
func deleteDuplicates(head *ListNode) *ListNode {
    var dupMap = map[int]bool{}
    // 用map记录重复值
    for p := head; p != nil; p = p.Next {
        _, ok := dupMap[p.Val]
        if ok {
            dupMap[p.Val] = true
        } else {
            dupMap[p.Val] = false
        }
    }
    // 删除重复节点
    left := &ListNode{}
    newHead := left
    for right := head; right != nil; right = right.Next {
        if !dupMap[right.Val] {
            left.Next = right
            left = right
        }
    }
    left.Next = nil
    return newHead.Next
}
```

## 复盘

啊！！没看清楚题目！！题目说了链表是已经排好序的。那么重新实现一下。

主要思路是，如果当前节点与下个节点重复了，那就把所有的重复节点都跳过去；如果不重复，就拼接到有效链表上。

```go
func deleteDuplicates(head *ListNode) *ListNode {
    left := &ListNode{}
    newHead := left
    for right := head; right != nil; right = right.Next {
        dupFlag := false
        for right.Next != nil && right.Val == right.Next.Val {
            dupFlag = true
            right = right.Next
        }
        if !dupFlag {
            left.Next = right
            left = right
        }
    }
    left.Next = nil
    return newHead.Next
}
```
