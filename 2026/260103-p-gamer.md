# 背景

近期中日关系紧张，在社交媒体上经常能刷到二战相关的内容，潜移默化中吸引了我的注意力。其中有一款游戏让我挺感兴趣的，看起来是类似文明那种世界地图模拟经营加即时战略的游戏，特色是有很详细的二战设定。我了解到这款游戏是《钢铁雄心4》，于是入了坑。

有一说一，P社的游戏我之前是玩过的，《群星》也是很经典而且相对更流行的作品。当时我玩过之后，虽然我有些种族洁癖，但是也没觉得自己有多战犯，一来可能是因为没有玩的很认真、即用冷血和利益的眼光做决策，二来毕竟都是外星人、更不容易让人建立与现实世界的联系，所以当刷到网上玩“P社战犯”梗的时候，我从没有觉得自己也是其中一份子。

# 不得不战的德国

关于德国为什么要开启二战，应该大多数人都会想起元首在演讲时讲到“一个面包50万马克”的画面，也就是犹太人把控了经济，通过垄断炒作最终酿成了巨大的社会矛盾，催生了极端的法西斯主义，最终点燃了世界大战。

但这其中还有很多细节，甚至可能出人意料。

德国在一战投降后，在1921年伦敦会议上签署了1320亿金马克的赔款。

这是什么概念呢，从几个方面分析：第一，当时德国年GDP大约500亿，1320亿金马克大约是2.64倍。换算成今天的中国，相当于赔款353万亿元人民币。第二，1901年中国签的辛丑条约可谓是近代屈辱的巅峰，赔款计算方式是“人均一两”，总计本金为4.5亿两白银，这就已经直接掏空晚清财政了。而按当时金价计算，500亿金马克约合14.8亿两白银，相当于德国“人均67两”。很显然，就算当时德国作为世界强国比晚清强大太多，这笔赔款也是绝对无法实现的天文数字，与意志无关，存粹是物理上无法实现。

如此离谱的赔偿条款，让德国国内非常不满。因为一战时德国是侵略方，战场都在法国本土，德国本土没有受到伤害；况且当时并不是真的战败了，而是德国政府感觉打不动了、主动选择和谈，对于国内一般民众来说，相当于是昨天报纸上还刊登着胜利，第二天就被告知每个人要还67两白银的巨额赔款。

德国政府面对这笔无法偿还的巨款，直接祭出终极大招：恶意货币贬值。要赔500亿马克是吧，那我开印钞机印大面值的钞票，一亿一张那只需要500张纸就行了对吧。理论上没问题，但是整个社会包括德国所有人都被牵连其中，货币贬值成废纸，只有实物资产才能保有价值，因此在工业化过程中成为工人、也就是无产阶级的大多数德国普通百姓成为了牺牲者，而持有土地、工厂等实物资产的德国贵族阶级（容克贵族）在这个过程中狠狠发了一笔国难财。

德国恶意贬值，那作为债主的法国自然不同意。但是法国在一战中“打光了一代人”，已经完全没有实力与德国抗争。于是在英美的调解之下，降低了赔款金额并且允许分期付款，甚至，英美为了“保持欧洲大陆的战略平衡”，主动向德国提供了大量的贷款帮助其恢复经济，这导致德国反而在1920年代迎来了一波小小的经济高潮。

然而好景不长，1929年至1933年之间发源于美国的大萧条来了，德国作为制造业大国也首当其冲。也正是在这样的背景下，元首开始在酒馆进行他精彩的演讲了，极端的法西斯主义正式统治了德国。

什么是法西斯主义？摘自百度百科：该主义以极端民族主义为核心，反对自由主义和共产主义，主张对内实行秘密警察统治与高压政策，对外推行武力扩张与民族压迫，其政治表现为集权主义，经济表现为统治主义，文化表现为极端主义。该主义被视为帝国主义的极端变种形式。

《凡尔赛条约》不仅仅是赔款，也有割地。因此对于极端民族主义的法西斯来说，夺回失地是必然的选择。而一旦开始扩充军备、用武力威慑夺回失地，那就一发不可收拾了。在游戏中感受特别明显，作为法西斯的德国，放眼望去在整个欧洲甚至整个世界范围，都是与自己意识形态不同甚至敌对的国家，从这点来看，希特勒寻求“生存空间”的说法其实是非常可以理解的。

《钢铁雄心4》游戏的剧本开启于1936年1月。这时候，意大利在打埃塞俄比亚，日本已经傀儡了中国东北，德国也撕毁凡尔赛条约开始征兵、出兵莱茵兰。这时期的英美主要是默许的态度，一方面也在应对经济危机后遗症，一方面也想搞制衡或者发战争财，并没有对法西斯进行实质打击。

1939年9月闪击波兰，第二次世界大战正式拉开帷幕。而在中国视角下，从37年7月7日卢沟桥事变开始，就算是二战了。

# if：经济奇迹

如果玩家选择扮演德国，进入游戏第一时间会发现，国家上有一个DEBUFF叫“梅福券”。它的机制，简单通俗来说，就是一种高利贷，初期可以让你的国家经济发达，快速建设，而随着时间推移，成本越来越高，高到国家经济瘫痪，再由对外战争掠夺来弥补。

在游戏中相关的政策叫“四年计划”，这是符合历史的。在历史上，它是一种政府担保的商业汇票，用于向军工企业支付购买军备，而德国政府承诺5年还本付息，也就是说这是一种变相地、能绕过英美监管的战争债券。梅福券在1934年推出，承诺5年内赎回，这可不就是“四年计划”吗，等五年之期已到，国家已经武装完毕，直接开启二战了，之后就不需要这种偷偷摸摸的做法了。

而游戏同样给玩家提供了另一条互斥的线路：经济改革——也就是立即废止并偿还梅福券。这会导致短时间内社会动荡，经济瘫痪，但是只要坚持一段时间持续推进政策、深化改革，逐渐就能恢复经济，甚至让经济变得比以前更好。

梅福券路线的优势是短时间爆发，为尽早开战做准备；而经济改革路线则是用阵痛换取长久的发展，消除隐患，后期经济非常强大。

当然我们知道，前者是史实，后者是架空。但后者也可能有后者的道理，毕竟德国确实在大萧条之前有过一段繁荣期，德国工业的底子是很扎实的。

那可能有人要问了：那假如是后者，既然经济已经改革好了，那为什么还要发起战争呢？

诶，还真的可以不用发起战争。你甚至可以走民主路线或者加入同盟国，与英美一起抗击法西斯和邪恶的共色阵营。当然，作为这款游戏来说，至少我觉得，走这种路线就很没有意思了。

那假如在某个平行宇宙的地球上，真的出现了这种假如，德国也是盟军的一方，欧洲大陆没有乱成一锅粥，会是好事吗？

容我畅想一下，我觉得不一定。至少站在中国的立场上，不一定是好事。因为如果欧洲和平的话，那列强们很有可能将精力放在瓜分东亚利益上，也许能帮中国击败日本，但却也会扶植腐败的民国，保留青岛、港澳、东北、台湾等租界，继承以往的屈辱赔款，成为实质上的列强傀儡吧。在东亚战场上红军还能跟日本的纸皮坦克和国民党的德械师菜鸡互啄，可要是换成正牌德国师，恐怕是不如直接投降吧。

# 偷工减料的军工

既然是二战背景，那自然少不了坦克、飞机、战舰。玩家在游戏中也需要设计自己的坦克、飞机、战舰，并安排生产线去生产合适的数量。

比如坦克，分为三种底盘：轻型、中型、重型（还有两栖、超重型等忽略），在此基础上玩家可以选择武器类型（榴弹炮、加农炮），装甲（防御），引擎（移动速度），还有各种配件（无线电、烟雾弹、油箱等）。不同设计的坦克会有不同的数值指标，也会对应不同的造价。

一个挺有意思的新手陷阱是：不要设计“高达”坦克。也就是尽可能把一辆坦克的配置堆到最豪华，同时也对应非常高昂的造价。

而老玩家们往往是非常抠门的，他们宁可承受一些数值惩罚，也要尽可能把造价压到最低。

其核心逻辑其实也很好理解。你一辆坦克造价13块，我一辆坦克造价8块，我的产能就将近是你的两倍；在前线我两个装甲师打你一个装甲师，一个在前面拖住你，另一个从侧翼包抄你后路吃饺子，这你怎么可能赢呢？至少在前中期，刚刚开始机械化，所有国家都供不应求的时候，数量一定优于质量。除非等到后期，经济非常发达，或者是在某些特定战场环境下战线很短，塞不下很多部队的时候，质量才会优于数量。

甚至军工企业在选择特质的时候，有一项特质叫“偷工减料”，效果为：造价-5%，攻击防御-5%，这个负面词语的名称反而成为了几乎所有老玩家的选择，非常反直觉。

尤其是对于我这种玩过类似《坦克世界》游戏的玩家来说，潜意识里就想要造出极致装甲、极致火力、极致机动的高达坦克，在战场上杀个七进七出、无人能敌。而这个游戏却当头一棒，告诉玩家，你就应该造纸皮装甲的机枪拖拉机，别在乎性能，越便宜越好。

什么？你问我士兵的生命安全怎么办？抱歉，在P社玩家的视角中，人命只是一串数字罢了，甚至是最廉价的资源之一，甚至不如工厂的产能重要。

不过呢，游戏设定毕竟是局限的，游戏中的战斗只是攻防数字的计算，因此数量可以发挥作用。然而在现实历史中，重装甲坦克是真有可能以一敌百的，敌方的炮弹打不穿我，那我就能一直无损杀敌，这也是苏德战争后期双方坦克越来越重的原因。

# 高达师与填线师

接着上面所说，玩家已经通过压缩成本，生产了一定数量的坦克。那如何将这些坦克分配给部队呢？

策略一：平均分配，每个师分1个装甲营（约40辆坦克），配9个步兵营和若干支援连，每个师大约1万余人。

策略二：重点分配，将所有坦克集中分配给少数师组成“高达师”，例如9坦克营+9步兵营，每个师大约1.5万余人，剩下全是填线师，只有5个步兵营约5千人。

不论是游戏还是现实，防守都是比进攻容易的。因此策略二更有效：用便宜的部队防守，拖住战线，再集合精锐部队撕开战线，快速包抄。只要被断了后路，没有补给且多面受敌，再强大的部队也会变得不堪一击。

甚至玩家还有一种更极端的玩法，对待这些填线师别说坦克了，连枪都不发（系统最低限制配备20%的武器）。这也是在游戏圈内比较有名的一个梗“五个人用一条枪”。

什么？你问没枪怎么守阵地？坚守！用人命去填！人死光可以，阵地必须保住！——参考文献：“我不要他的伤亡数字，我只要塔山”。

# 占领区的抵抗

好了，你现在已经掌握了战斗的玩法了，很顺利地攻城略地，占下了很大一块区域。

这时你会发现，你占领的土地上的矿山和工厂并没有按你期望地运转。当地百姓拒绝合作，甚至时不时搞破坏，导致你不得不动用许多人力和装备去组建治安部队，还只能获得少量的资源产出。

这个机制也让我很意外。尤其是对于我这种玩过三国类的游戏的玩家来说，在以往的三国类游戏中，军队占领一个地区之后，只要简单地修复、赈灾、安抚之后，很快就能变成自己势力的一部分。

但是仔细一想会发现这个机制很合理。毕竟在三国的世界中，大家都是中国人，对于中国老百姓来说，只是换了个当官的人罢了，没什么区别；可是在欧洲大陆上，不同的国家就意味着不同的民族和语言，谁能容忍外族的统治呢？清朝统治者也很难获得民心，必须依靠残酷手段镇压啊。

说起来欧洲大陆整体来看，面积够大，资源也挺丰富，为什么没能像中国那样完成统一，就是缺了一位始皇帝啊。读书时，教科书上重点说的是焚书坑儒，是他的负面印象，而直到人到中年，才逐渐领悟到他的千古功绩。

那在游戏中最有效的统治方式是什么呢？不能靠明着硬抢，要靠软的智慧。一种是选择某些国策（系统内的一种核心资源、只能随时间推进也就自然与其他国策互斥），获得地区的“核心”，大概就是用“自古以来”的那一套，合理合法、心服口服地让当地人加入我们成为自己人。另一种就是扶持傀儡政府，表面上你们国家还是独立的，还是由你们自己人管理着，但实际上这个傀儡政权会以极低的价格向宗主国出卖资源，并交出所有的军事权力。

这样一想，当今现实世界中的“美帝”，还真是个巨大的帝国，半个地球都是他的傀儡国。真正的独立国家才是少数，像俄罗斯好歹仗着苏联的余威，而中国这种能从一穷二白发展起来，可真是伟大且令人自豪的奇迹。

# 战略纵深

我们国家能保持自主，还有一个很重要的原因，在于战略纵深。

什么叫战略纵深？例如我是德国，我要打苏联，打了一年付出惨重代价，好不容推到列宁格勒-莫斯科-斯大林格勒一线了，然后把地图拖过去一看，好家伙，东边还有四分之三个苏联呢。不过这个例子可能不太恰当，因为苏联的核心还是在西边。

另一个更恰当的例子是非洲。我是德国，打西班牙打到直布罗陀海峡，西班牙投降了，然后我一看地图，海峡对面的非洲北部也有一块是我的（西班牙的）。这块区域，如果我要的话呢，旁边法属殖民地军队马上会进军占领；如果我不要呢，那多不爽啊，必须要。于是我把部队开到了对岸。然后我想，来都来了，而且欧洲这边暂时没有战事、下一步要准备登陆英国或者苏联了，这会有时间，我就把非洲打下来吧。

然后我就掉进了泥潭之中。确实我的部队节节胜利，但是打得很辛苦，非洲太落后了，没有基础设施提供补给，没有油我的坦克都开不了。而且，非洲太大了。我从北非打到南非，打了快一整年，才只占领了四分之一个非洲，却还不得不在极度缺乏补给、战线横穿整个非洲大陆的巨大困难之下继续作战。而当我的主力部队在非洲大陆耽误时间的时候，另一边的英国和苏联却在加速备战，等我终于下定决心撤军，回来之后看见的苏联军队已经比一年前强大了很多。可以说，我的这场非洲战役是失败的。

这就是战略纵深的作用。不仅土地大，而且贫穷落后，而且附带天气武器（非洲有丛林和酷热，苏联有积雪和严寒），纵使我有所向披靡的装甲部队，也难以取得战略目标。

类似地，如果站在美帝、或者说任意一位地球球长的视角，中国大陆这片土地都是几乎无法战胜的存在。

面对一个不听话的国家，第一反应可以是“打”。可是打又打不过，抗日战争和朝鲜战争已经证明了这一点，用游戏术语可以说，解放军陆军拥有开挂般的组织度、核心领土攻防加成。如果非要争这口气，举全国（全球）之力去打，也许在第一阶梯，也就是东北、华北、华东、华南这些平原、丘陵、沿海地区，能取得一些胜利；可接下来到山西、四川、贵州这些山区的时候，恐怕就又要陷入战争的泥潭了。再加上占领区抵抗增长快，占领的收益可能比不上成本，扶持傀儡国又可能被抵抗组织推翻……这怎么打嘛，鼠标给你你来玩！

武力打不过，那经济封锁呢？确实会给中国造成很大的困难，但是也就是苦一点、慢一点，终究还是能慢慢发展的，做不到只靠封锁海上交通就把中国封死。

还有一个不得不思考的问题，我作为帝国最高领袖，我的政治地位如何经营？与这样一个国家长期处于敌对状态，投入大量的资源，收益完全不成正比，那么我在国内的政敌或者反动势力会不会借题发挥？就算也许经过几十年的不懈进攻，最后终于彻底占领了这片大陆，这属于是苦了这代人却利好千秋百代的巨大功绩，但是为什么非得苦我这辈呢？我这辈子稍微保守点，在总统之位上享尽荣华富贵它不好吗，为什么要去啃这个硬骨头？

可以说，中国最强大之处是这份强烈的民族认同感，在这样一个幅员辽阔、地形复杂的区域上，十几亿人口居然认同自己是同一个国家，并且还是一个历史悠久的、带着优越感的古老帝国。除非是三体人来了，否则只靠地球人是无法彻底战胜这样一个国家的。

# 百年海军

“十年陆军，百年海军”这句话相信很多人都听过。在木制风帆战列舰时代，建造大型战列舰所需的木材真是物理意义上的需要百年时间。

而在近现代，制造钢铁战舰所需的时间和成本依然远远超过陆军或空军。

德国是陆权国家，1936年开局时只有一支三十多艘战舰（2新式战巡、2老式战列、4重巡、若干轻巡驱逐）的舰队和十几艘潜艇，而同期的英国海军有两百多艘战舰，其中甚至包括几艘航母。正常情况下德国是正面打不过就搞偷袭，造大量的潜艇去偷袭敌国的运输船。而在游戏中，我被英国封锁得很难受，我非要打造一支强力舰队跟英国硬碰硬，甚至反过来封锁英国，我该怎么做？

我得在36刚开局不久就研究航母所需的科技，到37年完成设计正式开工建造，尽管我已经偷工减料选择便宜的设计方案了，可等第一艘航母下水已经39年了，下水了还得小心翼翼地配合路基航空兵在近海行动，真正开始尝试反攻得到40年以后。相比之下，陆军从37年开始列装新式中型坦克，到38年9月闪击波兰时，已经能装备好多个装甲师，打造一支当时陆地上的无敌军团了。

从造价来看也是惊人。一艘航母的造价约1万块，一支混编舰队的总造价约十几万；而同时期一辆坦克造价8块，一个主力装甲师（2装甲营）的总造价也就1千多块，也就是说一支舰队的成本相当于100个陆军装甲师。这100个装甲师铺开来，就算英法苏美所有陆军一起围攻恐怕都不落下风，可海军一支舰队却是有可能在一场海战中就全军覆没的。

建设一支海军，需要很长时间的规划布局，更需要强大的国家工业实力支撑，只有大国才能玩得起。
