# 背景

今夕是何年？

也想不起为什么突然想玩这款游戏了。大概是前阵子玩钢铁雄心，一直在师级单位以上、在战略地图上进行宏观指挥，然后萌发了想要亲自披挂上阵的想法，于是就想起了这款游戏。

其实我很久以前就买了它，但是短暂试玩之后放弃了，主要是当时没能理解游戏机制和融入游戏设定世界中。

如今认真地玩了玩，感觉还是挺有意思的。

# 新手陷阱

既然是一款战略游戏，又可以招兵买马组建军队，后期甚至可以建立王国称霸大陆，那我想很多人肯定跟我一样，开局之后就一边游历（做贸易、做任务）一边招兵买马。

但是这样玩了一会儿之后，我惊讶地发现，我居然养不起我手下这几十号人了。尽管我把队伍的背包装得满满的，甚至还用表格记录各个城市商品的价格数据，辗转于各个城市之间，但是我靠贸易赚的钱居然不够发军饷。每到一个城市大约赚个千八百块，而每天的工资就要发一两百块，但凡在某个城市没赚到钱，我就要陷入经济危机之中。

虽然我感到非常地不解和不爽，但是这回我没有放弃，我认真思考其中的原因。

其实，如果带入现实中的话，其实这还挺符合现实的。玩家所扮演的角色，虽然说是游戏主角，但是在这片大陆上，不过是一个初出茅庐、名不见经传、甚至刚刚被劫匪杀了父母绑架了兄弟姐妹的平民青年男子罢了，为什么有资格养一支几十人的雇佣军队伍？在21世纪的现在，屏幕前的你，就算是一位资深程序员、高级白领甚至公司管理层，你就能（靠自己的钱）养得起几十个人吗？答案显然是否定的。

然后我去网上搜了攻略。开局玩法有很多，比较靠谱和轻松的一条路是当“铁匠”。也就是开局主要把角色属性点点在铁匠上，然后靠打造特定的配方（低成本高售价）来快速赚钱。

诡异的地方在于，如果你选择这条路当一个铁匠，那你需要长时间停留在城市中。那士兵对你不仅毫无作用，甚至还白吃白喝浪费你的钱。

那什么时候需要士兵/军队？需要保护或争夺势力资产的时候。

什么叫势力资产？一座城镇，或者一座城堡，以及它们附属的村庄。

也就是说，只有当你是一名拥有领地的领主，或者被王国雇佣成为雇佣兵，你才需要军队。

如果你只是一个铁匠、竞技场达人或者一个商队头子，你完全可以孤身仗剑走天涯，一人吃饱全家不饿，或者说，你的资产也不足以供养一支庞大而精良的军队。只有当你获得稳定收入或者积累了充足的存款之后，并且积累了足够的声望（家族等级），才有资格走上贵族的牌桌。

这是这款游戏对于新手，特别是带着战略类游戏思维来的新手玩家，最大的陷阱。

# 前期规划

在《钢铁雄心4》中，开局的规划特别重要，甚至会影响整局游戏。而在《骑马与砍杀2》中，也是类似，开局必须要做好规划。

主要需要规划的是玩家角色的加点路线。主要分两大流派，“上三路”和“下三路”。上三路对应的是近战攻击、远程攻击、体力耐力三大类，主打的是前线搏杀；下三路对应的是狡诈、社交、管理三大类，主打的是后台管理。

我个人推荐的规划是，前期主要加单手武器、步行、铁匠和流氓习气。有了这些属性点，玩家可以去不同的城市参与竞技场（获得声望），可以打铁赚钱，还可以做潜入或者越狱等任务并且后期增加战斗后掠夺的战利品数量（本质也是赚钱）。同时这些属性点也足够让玩家胜任步兵队长的岗位，为步兵提供战斗力加成。中期转型，全力点医疗（医疗大招加士兵生命值、以及战场减少死亡改成受伤）、管理，次要点统御、魅力、工程。

# 讨老婆

除了玩家主角的属性点之外，还要慎重规划的，是家族成员和伙伴。获得家族成员的唯一方法是讨老婆，别怀疑，你没看错，就是**讨老婆**。你自己讨一个，然后还要帮你哥哥（后期还有弟弟）讨一个。因为游戏世界观是个跟古代中国一样的典型的父权社会，嫁出去的女儿泼出去的水，讨来的老婆就完全属于你的家族，这样玩家可以专门去找那些属性很高的女性npc结婚，本质上相当于招募将领了。

推荐的结婚对象同样分“上三路”和“下三路”两个派别。上三路，也就是武将方向的，有巴旦尼亚的科林、斯特吉亚的阿波拉涅娅；下三路有瓦兰迪亚的古东赫尔达、赖因加达、斯特吉亚的安德鲁塔、巴旦尼亚的弗洛赖德。

我个人首推科林。除了她恐怖的属性值之外，她还年轻（19岁），漂亮（人物建模真不错）。

当然关于“漂亮”的标准，如果按日韩游戏的标准来看，整个骑砍2游戏里都没有一个女人能看的。这里说的漂亮只是相对其他npc而言的漂亮，同时也能勉强看得过去。科林人称暴力小萝莉，身材娇小，短卷金发，表情邪恶，虽然说“骑砍玩三年，母猪赛貂蝉”，但是这个建模我确实看着不错，越看越喜欢。

此外，科林对我而言还有一个很重要的理由：缘分。玩家前期走铁匠路线，那就要去找木材多的地方，那就必然是“肖农”。而“肖农”这座城市正是巴旦尼亚王国治下，巴旦王和他的萝莉公主科林就住在隔壁的城市“马鲁纳斯”。

想当年，当我在肖农打铁大半年打烦了，出门一趟打算散心，第一站走到马鲁纳斯，无意间瞥见这个表情邪恶、属性逆天的小萝莉，当时可真是惊为天人，一见钟情，当即就决定此“生”非她不娶。

我向她展开追求，我们多次约会，我们进行了深刻的灵魂对话，最终得到她的认可和一句“你快点再来看我”之后，我去寻找巴旦王。巴旦尼亚的战争从没有停止过，巴旦王也常年在外领兵出征。经过一番周折后我终于找到了巴旦王，向他提亲，慷慨的巴旦至高王提出了他的彩礼标准：7130第纳尔。这7130块钱的彩礼甚至还比不上我打造出的一把武器！此时，我知道，我此“生”不仅要好好对待巴旦公主科林，还要为巴旦王东征西战，抛头颅洒热血（关于这点后面有变故，暂且不提）。

在网上也有很多网友与我有类似的想法。在一条“你选择开局加入哪个国家”的讨论贴下，有一条高赞评论：“巴旦尼亚，单纯是因为巴旦王是我老丈人”，道出了大家的心声。

除了科林之外，第二个结婚人选是古东赫尔达，推荐用哥哥去娶。其实有一说一，古东赫尔达的父亲赫卡尔比巴旦王更加慷慨：（自由恋爱的话）彩礼只要498第纳尔，不但如此，嫁妆还附带6级甲、6级马、6级马甲这些在前期有钱都买不到的王室顶级装备。

第三人选赖因加达也是找赫卡尔提亲，虽然不是他的亲生女儿。

只靠结婚的话，人还是远远不够的，毕竟卡拉迪亚大陆上有大几十座城镇和城堡，每个地方都需要一个总督，因此作为一个伟大的国王和统治者，玩家的首要任务其实是：生孩子，或者更准确地说：生男孩子。自己、哥哥、弟弟都要结婚，婚后尽量创造条件怀孕生子，然后儿子成年后再去娶媳妇……如此经过四十年游戏时间，当玩家成为爷爷辈的时候，玩家的家族就可以有几十甚至上百人了，可以单个家族统治卡拉迪亚大陆了（笑）。此时这个世界上也不需要贵族了，所有败在你手下的贵族可以统统砍头，这样就再也没有人可以反抗你了（大笑）。甚至你可以只砍男性贵族的头，留下寡妇贵族改嫁给你的儿子或者孙子（好了请别再不当人了）。

大概是为了引导玩家不要走上如此变态的玩法，游戏给玩家设置了一个主线任务，任务有效时间是20年游戏时间，也就是子辈刚刚成年不久。

怀孕也挺真实。你不能让一对夫妻长期分居两地（例如哥哥带在身边练级、嫂嫂在城市当总督），你至少得让他们团聚几天，才能怀孕。~~怎么说得像是种猪似的呢。~~

而“同伴”，则是一种属性比较低，但是可以招募用来代替贵族使用的一种npc，主要用来给玩家前中期过渡用。

建议前期必备这些同伴：①高侦察的“追踪者”，②高工程的“工程师”，③高铁匠的“铁匠”（前期帮忙赚钱），④高医疗的“急救士”（用来带一支家族部队），⑤高下三路的“王子”（当总督）。其实核心思想就是招管理支援型人才、而非战斗型人才，包括后续培养自己家族的人也是给他们主要加下三路属性点。

# 婚姻与贵族

上面说到婚姻方面，其实大家应该可以看出来，我似乎非常有“代入感”地在玩这款游戏。

确实挺带入的，因为游戏本身做的挺“真实”的。

甚至让我有点受到“文化冲击”。其中的典型是关于中世纪的婚姻制度和贵族制度。

第一，虽然我长期以来的认知，是欧洲人非常开放（放荡），贵族们不论男女都玩得很大，可是在游戏里却惊讶地发现，他们居然是一夫一妻制度。不过仔细想想也合理，毕竟他们玩得再花，那也是老公老婆各自养各自的情人，没有小妾一说，也没有离婚一说，确实是非常标准的一夫一妻制度。

而这种制度性的保证，居然是由宗教（基督教）带来的。这也隐含了欧洲国家的一个无法调和的隐形结构性矛盾：王权与宗教权的对立，即王国只能控制军队和土地，而教会控制着文化习俗，二者势均力敌，长期对峙，没有一方能完全吞并另一方。

而在此基础上，“王国”也不等于“国王”，国王只是一个王国中最强的一个家族，大量的土地和军队都掌握在其他家族手中，且“私有财产不可侵犯”，所以同样是名义上的国家最高统治者，欧洲的国王只能像个大地主，其权势远远无法与中国古代的皇帝相比。

第二，他们没有“大家族”的观念。虽然我上面提到一种生很多孩子的玩法，但那本质上是我用中国人的思想在玩欧洲人的游戏，是扭曲的。在游戏设定中，嫁出去的女儿可是完全的泼出去的水，与娘家再无丝毫关系，甚至我回头与老婆娘家人宣战，都没有任何怨言。至于子孙后代，按欧洲中世纪贵族的规矩，除了嫡长子留下继承爵位和领地之外，其他嫡子都应该送出去当骑士，或者给一小块独立的领地分封出去，不会留在老头子家里，更别说孙子了。至于私生子那跟奴隶一样没啥地位，游戏里则根本不存在。

当然这里需要严肃纠正的是，历史中的欧洲贵族并没有像游戏里这么断绝关系。两个家族联姻之后，还是有很多联系的，联姻的本质依然是政治交易。一个成功的贵族新娘，应该像大使一样通过书信、金钱和社交活动维护一片巨大的社交关系网，为双方家族提供强大的助力。但是从他们“小家族”的生活习惯来看，与中国大家族相比，确实会少很多婆婆妈妈的生活琐事。

第三，他们的嫁妆丰厚得吓人。实力雄厚的大家族给新娘准备的嫁妆，在游戏里体现的就是穿在她身上的装备，这些装备各个顶级，一身装备折合市价大约要好几十万，而且有价无市，有钱都没地方买。甚至玩家戏称“超便宜买了一身顶级装备，还附赠一个老婆”。

这种现实也是符合历史的。欧洲贵族们为了嫁女儿往往可谓是倾尽所有。最典型的案例是1152年，埃莉诺与亨利二世结婚时，将她继承的法国西南部的庞大领土——阿基坦公国（占当时法国约一半面积）带入英格兰王国。而中小规模的贵族也同样不例外，我们后世可以从很多文学作品管中窥豹。甚至有些实力衰弱的贵族，如果准备不起符合地位的嫁妆，那为了不让女儿和自己家族蒙羞，直接选择把女儿送进修道院当修女，相当于中国的出家当尼姑。

主流观点认为，除了由于父母对女儿的爱、希望女儿在新家庭中不被丈夫轻视这类感情上的理由之外，更利益化的考虑是，巨额嫁妆是女方家族作为“嵌入男方家族的继承体系”所支付的成本。在中世纪欧洲，欧洲的贵族制度是纯粹按血统论的，并且女性也有同等的继承权（虽然往往不会直接成为领主，往往会被联姻带入丈夫家，但这继承权是实打实存在的）。而领主不只是享有领土权力，同时也负担着保家卫国的义务，贵族往往都是战士，死在战场上是家常便饭，而在基督教的影响下他们坚持一夫一妻制度，子女往往不会特别多，因此贵族家庭绝嗣的场景经常发生，而在这种场景下女方家族往往就能借助这种远方亲戚的血缘关系来主张继承权。

在继承方面，也是有着属于他们的智慧。长子继承几乎所有的家族产业，剩下其余的子女则离开家族，前往教会、商业、军队、文化领域自行创业。这些来自贵族的次子们往往是受过良好教育的精英人士（相对于平民/农奴来说），同时借助家族的力量，往往能够在新的领域中发光发热。用中国话可以说是“长子求稳，次子求变”，这样就给社会注入了创新和发展的动力。再加上欧洲大地上长期分裂，在中国视角下的“欧洲小国家”们，就在优胜劣汰的残酷自然选择之下，更有机会诞生出（相较于封建主义）更加先进的制度——资本主义。而同时期的中国，明、清两朝的统治者依然把“稳”作为重中之重，甚至主动拒绝新的科技和新的思潮，就这样错过了时代的机遇，导致了后来的百年屈辱史。

看看2026年的今天，随着巴以战争、俄乌战争、中美经济战等一系列事件的持续发酵，我们会发现“东升西落”的趋势越来越明显，我们中华民族正在加速且势不可挡地重新回到世界民族之巅，重新夺回我们历史上一直享有的大国地位。作为一名中国人，这固然是一件令人高兴的事，但同时也应该注意到，以我们国家（或者说中华文明）为代表的中央集权制度并不是完美的，在创新能力方面确实存在制度上的缺陷，这是值得警惕和反思的。

通过上面所说的这些，不难发现，中世纪的欧洲和古代中国的婚姻和贵族制度有着非常明显的不同。写到这里，我也深深感到自己知识之贫瘠。我想我需要再读一些相关的书籍，以及类似《十字军之王3》这样的游戏，来对这段历史更加深入的了解，再来重新修订本文所述的观点。这里也向大家推荐B站UP主“星白Knut”，他做了很多关于中世纪欧洲历史的视频，例如[贵族娶妻能大发财？中世纪欧洲的彩礼和嫁妆](https://www.bilibili.com/video/BV19T421e7tZ)。

作为一个未婚男性，如果我说我对这种近乎天上掉馅饼的嫁妆制度没有一丝一毫的心动，那显然是不现实的。但我也可以坚定地说，我也不是很支持这种制度，我觉得任何会让原本家庭难以承受的婚姻嫁娶习俗，不管是巨额彩礼还是巨额嫁妆，都是陋习。也许是理想主义，也许是幼稚，但我还是相信，理想的婚姻还是应该由夫妻二人一起亲手创造属于自己这个小家庭的财富，原生家族如果有余力，可以在这个过程中提供持续的帮助，而不是一次性投入巨大的成本。

# 爱兵如子

在骑砍2玩家群体中流传着一个梗：“骑砍新玩家：你们先在后面等着，等我上去搏杀一番；骑砍老玩家：F1+F3”（其中F1F3指的是游戏中的“全军冲锋”命令）

意思是说，玩家可能会非常“珍惜”自己手下的士兵，宁可自己孤身一人冲上去与敌人搏杀，也不愿意让士兵冒着生命危险去战斗。

除了“享受战斗”这个理由之外，对于玩家来说，还有一个很现实的理由：培养高级士兵真不容易，高级士兵死了会心疼。

这倒是让人哭笑不得。上个月我还是把人命当作数字和最廉价资源的P社战犯，这才多久，我就变成了爱兵如子（甚至爱兵如父）的T社好领主。

如果新兵已经征召进入了自己的队伍中，只需要让他积累经验值升级，那还好说。可是最痛苦的点在此之前的征兵环节——玩家必须亲自前往各个村庄，一个村一个村地招募新兵。

这在一定程度上体现了当时欧洲的军事征兵制度——几乎所有村庄的所有村民都是正在屯田的士兵。他们平时务农，当村庄遇到危险、或者国王或高级领主前来征召的时候，他们就去铁匠铺拿起武器，成为士兵。这种画面有些过于接地气了，也难怪中国网友们都调侃欧洲贵族之间的战争就跟“村斗”似的。这种全民尚武的风气，甚至比汉人更加夸张。

这同时也反应了《骑砍2》这款游戏的一个最严重缺陷：玩家几乎无法将工作委托给手下的将领或者封臣，就连征募新兵这种小事都必须亲历亲为，使得游戏中后期的游戏体验非常差。

所以从这一点出发，如果你想玩的是亲自驾驭战马、手持偃月刀冲进步兵阵列中痛快地砍杀，那你可以玩《骑砍2》；而如果你想体验经营和指挥的乐趣，那应该《全面战争：三国》或者《钢铁雄心4》会更适合你。
